Проєкт Greenhouse реалізує цикл роботи з XML-документом, що описує рослини оранжереї: створено XML та XSD зі складними й простими типами, перерахуваннями, шаблонами, граничними значеннями й атрибутами типу ID; згенеровано відповідні Java-класи та побудовано додаток, який аналізує XML за допомогою DOM, SAX і StAX парсерів та ініціалізує колекцію об’єктів. Додано сортування рослин за допомогою Comparator, перевірку XML на відповідність XSD, а також XSLT-трансформацію XML у HTML. Увесь функціонал покритий юніт-тестами, а проєкт оформлено як повноцінний Maven-додаток.
